Ejercicios Etiquetas Interactivas

Details

Instrucciones de uso

Instrucciones

  1. Summary es una ETIQUETA dentro de Details
  2. El contenido de Summary es el título del desplegable Details
  3. Atributo Open para que aparezca desplegado

Dialog

Instrucciones

  1. Dialog es un "PopUp"
  2. Dialog necesita el atributo id="nombre"
  3. Los botones se referenciaran al dialog por el id
  4. Necesita el atributo Open para que se VISUALICE

Popover

Instrucciones

  1. Popover es un ATRIBUTO
  2. Se puede asociar a otras etiquetas
  3. Los botones se referenciaran a la etiqueta por el id

Datalist

Instruciones

  1. Input de texto con selector de opciones
  2. El usuario puede escribir cualquier cosa
  3. Despliega opciones al clickar la flecha al poner el cursor encima
  4. El Input NECESITA el ATRIBUTO list
  5. Que se vincula al ATRIBUTO id del Datalist
